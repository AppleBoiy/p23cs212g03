A prerequisite tree is a visual representation of the courses that a student must complete before taking a particular course. It shows the relationships between courses and helps students plan their academic path.

The Prerequisite Tree tool allows users to visualize and explore the prerequisite relationships between different courses. It fetches course data from a Google Sheets, converts it into a JSON format, and generates a tree view using the dagreD3 library. Users can hover over nodes to view prerequisite details and navigate the tree to understand the course dependencies.

Pre3 is a course management system that helps students and faculty manage courses, prerequisites, and degree requirements. It provides a visual representation of course prerequisites, helps students plan their academic path, and allows faculty to manage course data and prerequisites.
